What is a DFA?

A Deterministic Finite Automaton (DFA) is a theoretical model of computation. It consists of a finite set of states and transitions between them based on input symbols.

  • Deterministic: For every state and input symbol, there is exactly one transition.
  • Finite: There are a limited number of states.
  • Automaton: It works automatically once an input string is provided.
